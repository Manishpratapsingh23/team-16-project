*Book Swap & Lending Platform

        A community-driven web platform where users can list, lend, swap, and borrow books in real time. Built with React + Vite on the frontend and Node.js + Express + Socket.IO on the backend.

*Features
User / Reader

        Register & create a personal book library

        Add, edit, or delete books

        Mark books as available for swap, lending, or donation

        Search books by title, author, genre, or location

        Send borrow/swap requests

        Track request status & due dates

        Receive real-time notifications

*Book Owner

        Approve or reject borrow and swap requests

        Update book availability

        Set borrowing preferences (friends / verified / all users)

*Admin

        Manage users & books

        Handle disputes or reported content

        View analytics: most-borrowed books, top contributors, trends

*Tech Stack

        Frontend
            React + Vite, React Router, Axios, Socket.IO Client, TailwindCSS

        Backend
            Node.js, Express, Socket.IO, JWT Authentication, MongoDB + Mongoose

*Infrastructure

        Redis (caching + notifications)
        Deployment: Vercel+render
